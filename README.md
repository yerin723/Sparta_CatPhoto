# 🖥️ Sparta Cat Photo
Sparta Coding Club 인생사진 쏙쏙 코딩네컷 강의를 수강하면서 만든 간단한 고양이 인생네컷 제작 웹 페이지입니다.

## 🔎 주요 기능

### 동영상 삽입 기능
가장 마지막 컷에는 YouTube '뚱치즈' 동영상이 삽입되어 있습니다. 

### 마우스 오버 효과 & 클릭 효과
고양이 사진 프레임에 마우스를 올리면 텍스트가 표시되고, 클릭하면 해당 사진의 번호가 알림창으로 출력됩니다.
